This repository contains a comprehensive web-based Proof of Concept (POC) for YORI, a mobile journaling application designed for Gen Z. It serves as an interactive mockup to demonstrate the core value proposition, user flow, and aesthetic "vibe" of the V1 product before full mobile development.

üå∏ About The Project
YORI is a Zen-Gacha Journaling App. It combines the therapeutic benefits of journaling with the fun of collecting digital stationery (stickers, papers, washi tapes) created by human artists.

Key Features Demonstrated in this POC:

The Daily Ritual: Opening a digital "envelope" to receive daily rewards (gacha mechanic without monetization pressure).

Digital Scrapbooking: A tactile editor to write thoughts and decorate pages with stickers.

Wellness Tracking: A "Pixel Year" mood tracker and emotional insights.

Collection System: A "Pokedex" style gallery for collected assets.

Ethical Stance: Simulates an ecosystem based on 100% human art (No AI).

The final product (Features are simulated for demonstration purposes).
